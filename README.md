# 💰 FundFlow – Digital Banking System

FundFlow is a backend banking application built using **Spring Boot** that enables users to manage accounts, transfer funds, and view transaction history securely. It follows RESTful principles and includes secure authentication and role-based authorization.

## 🚀 Features

- 🔐 User registration and login
- 🛡️ Role-based access control (Admin/User)
- 💸 Secure fund transfer between accounts
- 🧾 View transaction history
- 💰 Check account balance
- 📊 RESTful APIs built with Spring Boot

## 🛠️ Tech Stack

- Java 22
- Spring Boot
- Spring Security
- MySQL + Spring Data JPA
- Maven
- Postman (for API testing)

## 🔮 Future Improvements

- 🔐 JWT-based authentication (token issuance, refresh, expiry handling)
- 📱 Frontend dashboard using React 
- 📈 Exportable transaction reports (PDF/CSV)
- 🔔 Email notifications for transactions
- 📜 Swagger/OpenAPI documentation
